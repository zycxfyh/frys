# 贡献者公约

## 我们的承诺

作为贡献者和维护者，我们承诺让社区参与对每个人来说都是无骚扰的体验，无论年龄、体型、可见或不可见的残疾、民族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性认同和取向如何。

我们承诺以促进开放、包容、多样化、包容和健康的社区的方式行事和互动。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性关注或挑逗
- trolling、侮辱性/贬损性评论以及个人或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如物理地址或电子地址
- 其他在专业环境中可能被合理视为不适当的行为

## 执行责任

项目维护者负责澄清和执行可接受行为的标准，并对任何不可接受行为的实例采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝评论、提交、代码、wiki 编辑、问题和其他不符合本行为准则的贡献，并将在适当情况下传达审核决定的原因。

## 适用范围

本行为准则适用于项目空间和公共空间，当个人代表项目或其社区时。

代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体账户发布，或作为指定代表在在线或离线活动中行事。

## 执行

具体执行政策可能在此处进一步详细说明。
