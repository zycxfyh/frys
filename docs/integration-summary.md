## 🚀 使用指南

### 开发环境
```bash
# 快速启动开发环境
pnpm install
pnpm run light:start

# 插件热重载开发
# 编辑 plugins/ 目录下的文件，系统自动重载
```

### 生产构建
```bash
# 选择最适合的构建方式
pnpm run light:build:swc    # 快速构建
pnpm run build:prod         # 完整生产构建
pnpm run build:multiarch    # 多架构构建
```

### 部署选项
```bash
# 传统部署
pnpm run docker:build
pnpm run docker:push

# 多架构部署 (推荐)
# 推送到main分支，自动触发多架构构建和发布
git push origin main
```
